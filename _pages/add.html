How to add you aliases to the list

  • Create a new draft Here
  • add your title, author information, categories, tags as appropriate
  • After the Front Matter add your alias then the More tag and a brief description on the function of the alias and how to use it if applicable.
  • commit the file, and the system admins will be notified and will publish your post

How to add aliases to your profile

  • Open the Terminal, and make sure you are in the root of your user folder.
  • Type nano .bash_profile
  • you may now add your changes and aliases
  • to save press ctrl+o, Enter, then exit by ctrl+x
  • Activate changes by typing source .bash_profile
